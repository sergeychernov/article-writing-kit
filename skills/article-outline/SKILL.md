# Article outline

Help the user plan a clear article before writing.

## Workflow

1. Ask for topic, audience, and desired length if not provided.
2. Propose a title and 3–7 section headings with one-line summaries.
3. Suggest where examples, data, or quotes would strengthen each section.
4. Offer an optional hook for the introduction and a closing call to action.

Keep outlines scannable — bullets over paragraphs.
