# Article Workspace Instructions

When working with articles in this repository, follow the Obsidian-friendly article rules in `.cursor/rules/article-writing-obsidian.mdc`.

Use this repository as a local Markdown workspace:

- One folder per article.
- `index.md` is the publishable entry point.
- `lead.md` and `act-*.md` contain publishable prose.
- `three-act-outline.md` contains structure, editorial intent, and working notes.
- `images/` contains screenshots, diagrams, and other visual assets.

Recommended workflow:

1. Use `article-scaffold` to create a new article folder.
2. Use `article-brief` to clarify topic, audience, pain, promise, and boundaries.
3. Use `article-architect` to fill `three-act-outline.md`.
4. Use `act-writing-coach` to write one act at a time.
5. Use `publish-prep` before publishing.
